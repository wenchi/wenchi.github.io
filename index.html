    關於 iOS 的記憶體管理 ( Reference Counting )

    Reference Counting 是記憶體管理的方式，將該物件的參照次數儲存起來，每次該物件被參照到，就將參照次數 +1，被釋放時就 -1，參照次數變成 0 時，立刻移除物件，該物件佔的記憶體會被歸還，有基本的觀念後，接著就分為手動管理記憶體(MRC)與自動管理記憶體(ARC)。
